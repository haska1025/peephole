# peephole

### 业务模型

```c
+---------+ 1      N +----------+ N       1 +------------+ 1    N +---------------+
| Session |--------->| Channel  |<----------| User       |------->| TcpConnection |
+---------+          +----------+           +------------+        +---------------+
```
#### 业务模型说明

User: 可以观看视频，也可以共享视频

Session: 当有一路视频的时候创建一个session，需要共享的人需要申请共享；需要观看视频的人，需要申请观看。

Channel: 是一个视频的逻辑通道，当一个人申请去共享、观看一个Session的时候会创建一个Channel。

TcpConnection：是tcp连接的抽象。
